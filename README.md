# Utah Division of Water Rights Data Models

The Utah Division of Water Rights collects, stores, and manages water use data from locations and organizations throughout the state of Utah. This repository contains designs and documentation for information and data models used by DWRi's database and software systems to represent the data they manage. Data models for the following datasets are represented in this repository:

* **[Water Measurement Data](measurement_data/)**: Water use data measured at diversion locations, within Distribution Systems, and at other locations primarily consisting of flow data and storage volumes.
* **Water Use Program Data**: Water use and water diversion data self reported by municipal, industrial, and other water users throughout the state.
* **Distribution Accounting Data**: Data produced by the distribution accounting process.
* **Water Rights Data**: Information and data about water rights within Utah.

Each folder in this GitHub repository contains documentation for one of the datasets listed above.
